## ğŸ’¼ **For DIET Coordinators**

### **How to Create a Cohort**

1. **Navigate to Cohorts**
   - Click "Cohorts" from the dashboard
   - Click "Create New Cohort"

2. **Select Needs**
   - View all reported needs in your district
   - Filter by:
     - Block
     - Grade band
     - Subject
     - Issue type
     - Infrastructure level
   - Select needs to include (or use AI suggestion)

3. **Use AI Assistance** (Recommended)
   - Click "AI Generate Cohort"
   - AI analyzes all needs and suggests:
     - Which needs to group together
     - Cohort name
     - Primary issues
     - Infrastructure level
     - Language preference
     - Estimated teacher count

4. **Review & Customize**
   - Edit cohort name if needed
   - Add/remove needs manually
   - Adjust description
   - Set tags for easy discovery

5. **Save Cohort**
   - Click "Create Cohort"
   - Earn 25 points! ğŸ‰
   - Cohort is now ready for plan generation

---

### **How to Generate a Training Plan**

1. **Select a Cohort**
   - Go to "Cohorts" section
   - Click on a cohort you created
   - Click "Generate Training Plan"

2. **AI Plan Generation**
   - AI analyzes cohort profile:
     - Primary issues (e.g., FLN gaps, language mismatch)
     - Infrastructure level (LOW/MEDIUM/HIGH)
     - Language preference
     - Grade band
   - Scores all 40 training modules
   - Selects top 4-6 most relevant modules

3. **Review Generated Plan**
   - See selected sessions:
     - Session title
     - Learning objectives
     - Duration (2-4 hours each)
     - Module details
     - Trainer notes
   - Total plan duration displayed

4. **Customize Plan** (Optional)
   - Reorder sessions by dragging
   - Remove sessions (click X)
   - Add different modules (click "Add Session")
   - Edit trainer notes for context

5. **Save as Draft or Publish**
   - **Save Draft**: Keep editing later
   - **Publish**: Lock plan, ready for training
   - Earn 50 points for generating! ğŸ‰

## ğŸ†˜ **Troubleshooting & FAQs**

### **Common Issues**

**Q: I can't login**
- A: Clear browser cache and cookies
- Check if you're using the correct email
- Try "Forgot Password" to reset

**Q: Voice input not working**
- A: Allow microphone access in browser settings
- Check if microphone is working in other apps
- Try using Chrome or Edge browser

**Q: WhatsApp message not getting response**
- A: Check if you sent to correct number
- Wait 1-2 minutes for AI processing
- Message should be in text format (not voice note)

**Q: AI generated wrong cohort**
- A: You can manually edit cohort
- Remove/add needs as needed
- AI learns from your edits

**Q: Can't find a specific module**
- A: Use "Learning Network" search
- Filter by category, grade, subject
- Contact DIET if module is missing

**Q: Points not updating**
- A: Refresh page (F5)
- Points update in real-time but may take 1-2 seconds
- Check "Gamification" page for detailed breakdown
